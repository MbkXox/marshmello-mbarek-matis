# marshmello

## Fonctionnalités

### Projets

- [ ] Créer un projet
- [ ] Renommer un projet
- [ ] Supprimer un projet (et toutes les tâches associées)

### Statuts

- [ ] Créer un statut
- [ ] Renommer un statut
- [ ] Supprimer un statut (et déplacer les tâches dans un statut "non classé")
- [ ] Changer l'ordre des statuts

### Tâches

- [ ] Créer une tâche
- [ ] Modifier une tâche
- [ ] Supprimer une tâche
- [ ] Déplacer une tâche d'un statut à un autre
- [ ] Toggle une tâche (terminée/non terminée)

### Utilisateurs

- [ ] Créer un compte
- [ ] Se connecter
- [ ] Se déconnecter
- [ ] Supprimer un compte
- [ ] Modifier ses informations
